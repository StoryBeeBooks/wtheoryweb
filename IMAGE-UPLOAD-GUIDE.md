## Image Specifications

### Dimensions
- **Recommended:** 1200 x 800 pixels (3:2 aspect ratio)
- **Minimum:** 800 x 533 pixels
- **Maximum:** 2000 x 1333 pixels

### File Format
- **Preferred:** JPG (for photos/screenshots)
- **Alternative:** PNG (for graphics with transparency)
- **Not recommended:** GIF, BMP, TIFF

### File Size
- **Target:** 200-400 KB per image
- **Maximum:** 500 KB
- **Tip:** Use image compression tools like TinyPNG or Squoosh to reduce file size

### Quality Guidelines
- Take full-page screenshots of the live websites
- Capture the hero section or most impressive part of each site
- Ensure text is readable (not blurry)
- Use good lighting/contrast
- Avoid watermarks or overlays

## How to Capture Website Screenshots

### Option 1: Browser Extension (Recommended)
1. Install "Full Page Screen Capture" or "GoFullPage" extension
2. Visit the website
3. Click the extension icon
4. Download the screenshot
5. Crop to focus on the best parts (if needed)
6. Resize to 1200x800px using an image editor

### Option 2: Built-in Browser Tools
**Chrome/Edge:**
1. Press F12 to open DevTools
2. Press Ctrl+Shift+P (Cmd+Shift+P on Mac)
3. Type "screenshot" and select "Capture full size screenshot"
4. Crop and resize as needed

**Firefox:**
1. Right-click on the page
2. Select "Take Screenshot"
3. Choose "Save full page"
4. Crop and resize as needed

### Option 3: Online Tools
- https://www.screely.com/ (adds browser frame)
- https://screenshot.guru/
- https://www.site-shot.com/

## Testing

After adding images:
1. Refresh the website in your browser (Ctrl+F5 or Cmd+Shift+R)
2. Check that all images load correctly
3. Test hover effects (images should zoom slightly on hover)
4. Verify mobile responsiveness (images should stack vertically)
5. Check loading speed (should be under 3 seconds)

## Troubleshooting

**Image doesn't appear:**
- Check filename spelling (case-sensitive)
- Verify file is in `Image Assets/` folder
- Clear browser cache (Ctrl+Shift+Delete)
- Check browser console for errors (F12)

**Image is distorted:**
- Ensure aspect ratio is close to 3:2 (width:height)
- Try uploading a different size (1200x800 recommended)

**Slow loading:**
- Compress images using TinyPNG
- Reduce image dimensions if too large
- Consider WebP format for better compression
